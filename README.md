# healthcare-analytics-powerbi-dashboard 

A comprehensive Healthcare Analytics Dashboard built using Power BI to analyze patient outcomes, drug effectiveness, treatment duration, dosage patterns, and reported side effects. The dashboard provides interactive insights through KPIs, bar charts, and scatter visualizations, enabling data-driven decision-making for healthcare analysis. It includes dynamic filters for condition, drug name, gender, and side effects, along with summarized insight cards highlighting key trends and outcomes.

Tools & Skills: Power BI, Data Modeling, DAX, Data Visualization, Healthcare Analytics
